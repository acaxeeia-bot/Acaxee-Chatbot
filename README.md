# Acaxee Chatbot — CRM & WhatsApp Agent

Sistema de atención al cliente, reservaciones y atribución META para Acaxee Sky Tropic Lounge, Mazatlán.

## Stack
- Node.js + Express (agente WhatsApp)
- Supabase (base de datos)
- Claude API (inteligencia conversacional)
- Next.js (dashboard)
- Meta WhatsApp Business API

## Archivos
- `index.js` — agente principal y webhook
- `schema.sql` — estructura de base de datos
- `.env.example` — variables de entorno necesarias
